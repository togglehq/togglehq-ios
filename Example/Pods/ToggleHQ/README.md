## Installation

ToggleHQ is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod "ToggleHQ", :git => "https://github.com/togglehq/togglehq-ios-cocoapod.git"
```
